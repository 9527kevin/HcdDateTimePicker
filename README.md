# HcdDateTimePicker
 A beautiful DateTimePicker.

##Usage
You can clone this project by git.
```
git clone https://github.com/Jvaeyhcd/HcdDateTimePicker.git
```
Then you should copy the **HCDDateTimePickerView** folder into your project.
